实现原理

这个重叠卡片布局主要通过以下 CSS 属性实现：

  • display: flex - 创建弹性容器
  • margin-left: -40px - 负边距使卡片重叠
  • z-index - 控制卡片的层叠顺序
  • transition - 添加悬停动画效果
.card-container { display: flex; } .card { margin-left: -40px; /* 卡片重叠的关键 */ transition: all 0.4s ease; z-index: 1; } .card:hover { transform: translateY(-20px); z-index: 10; /* 悬停时提升到最上层 */ }
